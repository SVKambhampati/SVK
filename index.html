EnvisionVR

Overview

EnvisionVR is a 3D visualization tool designed to help special‐needs students improve spatial processing through immersive VR experiences. Built with Unity and deployed on Oculus Quest, it presents interactive environments that adapt to each learner’s pace.

Key Features

  • Immersive 3D scenes that teach geometry concepts
  • Customizable difficulty levels & adaptive feedback
  • Support for hand‐tracking controls (no controllers needed)
  • In‐app analytics dashboard showing user progress over time

Technologies

Unity · C# · Oculus SDK · Firebase · Blender (3D assets)

Challenges & Learnings

Integrating hand‐tracking on the Oculus Quest presented a learning curve—ensuring that interactive objects responded smoothly without latency was challenging. I ended up optimizing textures and mesh colliders to maintain 90 fps, and I learned how to profile performance in Unity for VR builds.
