EnvisionVR

A virtual reality platform built to support special-needs students with visual-spatial challenges — transforming traditional lessons into immersive 3D learning experiences.

Unity C# Meta SDK XR 3D Modeling
GitHub ↗

Overview

EnvisionVR is a VR education platform developed to assist special-needs students who struggle with visual-spatial reasoning. The application delivers interactive 3D environments that enhance cognitive development and conceptual understanding through immersive, hands-on learning experiences across a range of subjects.

Key Subject Modules

  • Biology: A guided human anatomy module where students can explore detailed skeletal and organ systems through interactive modeling.
  • Geography: An explorable Earth globe with dynamic topographical maps and real-time environmental simulations.
  • Physics: A virtual physics lab for conducting experiments like free-fall kinematics with live motion tracking and visualizations.
  • Astronomy: A 3D solar system where students can interact with rotating planets, moons, and orbits to learn about space in an engaging way.

Technologies Used

  • Unity — game engine powering all 3D environments and real-time simulations
  • Meta SDKs — for headset integration and hand/controller tracking
  • C# — application logic, UI controllers, and shader programming
  • Custom 3D modeling & shaders — environment art and visual effects
  • Real-time motion tracking & dynamic simulation

Impact

This project demonstrates how technology can be harnessed to bridge educational gaps, offering transformative tools that make learning more engaging and accessible for all students. EnvisionVR shows that immersive tech isn't just for entertainment — it can be a genuine force for inclusion.
