Project Three

Overview

Project Three is a machine‐learning‐powered image classifier that can identify different species of plants from photos. I built it using Python, TensorFlow, and a Flask API to serve predictions.

Key Features

  • Convolutional Neural Network (CNN) trained on a custom dataset of 10,000 plant images
  • Upload a photo in the browser and get instant classification results
  • Real‐time confidence score and top‐5 guesses
  • Mobile camera integration so you can take a photo directly from your phone

Technologies

Python · TensorFlow · Keras · Flask · HTML/CSS/JavaScript · Heroku Deployment

Challenges & Learnings

The biggest challenge was fine-tuning the CNN to avoid overfitting on my dataset. I ended up using data augmentation (rotations, flips) and dropout layers, which improved generalization significantly.
