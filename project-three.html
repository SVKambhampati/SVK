AddressSync Automation

Overview

AddressSync Automation was developed as part of a software engineering internship with DeKalb County Government. The goal was to enhance operational efficiency by addressing a critical issue: failed FedEx shipments due to mismatched or incomplete shipping addresses across Oracle systems. The solution automatically syncs address data between Oracle E-Business Suite (EBS) and Oracle Commerce Cloud (OCC). It detects discrepancies in shipping addresses for customer orders, retrieves accurate information from EBS, and updates OCC via REST API calls. This automation replaces error-prone manual correction and significantly reduces shipping failures. Additionally, the internship involved building an automated FTP pipeline to transfer nightly budget data from Oracle EBS to Oracle PBCS (Planning and Budgeting Cloud Service), increasing reliability and reducing administrative overhead.
