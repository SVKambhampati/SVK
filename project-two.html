Project Two

Overview

Project Two is a full-stack web application that allows users to track their daily habits and visualize progress over time. It features a responsive React front end, an Express/Node.js back end, and stores data in MongoDB.

Key Features

  • User registration, login, and secure JWT authentication
  • Create, edit, and delete habits (e.g., “Drink Water,” “Exercise”)
  • Interactive charts showing weekly/monthly habit completion rates (using Chart.js)
  • Mobile-friendly design so you can log habits on the go

Technologies

React · Redux · Node.js · Express · MongoDB · Chart.js · CSS Modules

Challenges & Learnings

While integrating Chart.js with Redux-managed state, I learned how to memoize data transformations so that charts re-render efficiently only when the underlying habit data changes.
