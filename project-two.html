MovieMind

Overview

With so many films released every year, picking the right one can be overwhelming. MovieMind solves this by learning your preferences and recommending movies tailored to your tastes. Whether you’re into sci-fi thrillers or indie dramas, MovieMind finds your next favorite flick.

Project Specifications

  • Recommendation Methods:
    • Content-Based Filtering: Analyzes movie metadata (genres, keywords, cast) to suggest similar films.
    • Collaborative Filtering: Leverages patterns in user ratings to find movies enjoyed by like-minded viewers.
    • (Optional) Hybrid Filtering: Blends both techniques for enhanced accuracy.
  • Features:
    • Search, rate, and discover movies
    • Personalized recommendations
    • Trending picks for new users
  • Dataset: MovieLens 100K
  • Modeling: TF-IDF for content similarity, SVD for collaborative filtering

Technologies Used

  • Languages: Python, HTML, CSS, JavaScript
  • Frameworks: Flask or FastAPI (backend), Bootstrap or Tailwind (frontend)
  • ML Libraries: scikit-learn, Surprise, pandas, numpy
  • Database: SQLite or PostgreSQL
  • Deployment: Render, Vercel
